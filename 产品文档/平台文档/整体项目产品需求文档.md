# 深圳市科技创新局国际科技信息中心 — 整体项目产品需求文档（PRD）

> **项目编号**：SZCG2025001150  
> **预算金额**：4.627亿元  
> **版本**：v1.0 | **日期**：2026-03-04  
> **甲方**：深圳市科技创新局  

---

## 一、产品愿景

> **打造全球首个政府级AI原生科技创新全要素智能平台** — 一个让深圳市科技创新局能够"看见全球科技全景、洞察创新趋势脉络、精准配置科技资源、高效推动成果转化"的数字底座。

### 与传统信息中心的根本区别

| 传统科技信息中心 | 本项目 |
|:---:|:---:|
| 文献数据库 + 统计报表 | AI原生情报中枢 + 决策引擎 |
| 人找数据 | 数据找人（主动推送） |
| 事后统计 | 实时感知 + 预测预警 |
| 各系统孤立 | 四大板块有机融合 |
| 人工分析报告 | Agent 30分钟自动生成 |
| 静态展示 | 交互式多维分析 |

## 三、板块一：智能服务集群 — 面向用户的一站式科技创新服务矩阵

### 3.1 智慧服务门户系统

#### 标书要求
智能搜索（全文/AI/关键词）、智能推荐（8类资源）、智能阅读（PDF增强解析）、智能科研（工作台/空间管理）、集群服务入口、系统管理、权限管理。

#### 🔥 创新价值点

| 创新特性 | 描述 | 竞品对比 |
|----------|------|----------|
| **AI科研工作台** | 不只是搜索入口，而是一个集文档编辑、数据表格、存档管理于一体的在线科研环境——科研人员"进门就能干活" | AMiner/CNKI仅为搜索入口 |
| **一码通服务集群** | 门户统一入口直达7大子系统，单点登录+统一权限，避免"切系统找功能"的体验割裂 | 行业内无此集成 |
| **智能埋点分析** | 全链路用户行为追踪，自动分析用户关注的科技领域和使用偏好，反哺推荐系统精度 | 竞品无平台级埋点 |

#### 🔥 创新价值点

| 创新特性 | 描述 | 政府决策价值 |

| **政策效果推演引擎** | 不只是展示政策内容，更能基于历史数据推演政策实施后对人才/产出/资金的影响 | 政策出台前先"模拟演练" |

| **科技时空演进可视化** | 用时间轴+地理空间双维度，动态展示深圳乃至全球的科技创新演化历程 | 讲好深圳科技创新故事 |

#### 🔥 创新价值点

| 创新特性 | 描述 | 政府决策价值 |
|----------|------|-------------|
| **科技需求→资源自动匹配** | 企业提出技术需求，系统自动匹配合适的科技人才/团队/机构，并推荐最优合作方案 | 政府牵线搭桥效率提升百倍 |
| **发展预测引擎** | 基于时间序列+图挖掘算法，预测人才/团队/机构未来3年的发展轨迹 | 科技投入的"前瞻性布局" |

| **决策全流程自动化** | 从任务分派→智能分析→报告生成→决策推荐→流转审批→结果归档，全链路自动化 | 科技管理"一键流转" |

### 第二轮：政府决策者感知 ✅

| 甲方关心的问题 | 本项目回答 |
|-------------|-----------|
| "建完之后谁来用？" | 600万注册用户目标 + 科技竞赛/直播/传播持续引流 |

| "数据安全怎么保证？" | 可信数据流通+隐私计算+联邦学习+政务等保三级 |
| "能不能用起来？" | AI数字人实训+智能问数+Agent对话，大幅降低使用门槛 |

### 第三轮：竞品不可替代性 ✅

> **核心结论**：全球没有任何一个竞品能覆盖本项目超过30%的能力维度。要达到同等能力，需至少采购15+个独立系统并进行深度集成——成本将超过本项目5倍以上，且无法实现数据融通和统一体验。
