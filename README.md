# 亿级科技知识图谱系统 (Knowledge Graph System)

## 项目简介
这是一个基于文本挖掘与大模型的亿级科技知识图谱系统原型。旨在打通“学术界”与“产业界”的数据壁垒，实现学术人才与产业链商业网的跨域融合。

## 核心功能
1. **AI 智能问答与报告生成**：通过自然语言搜索，自动生成深度研究报告。
2. **多维关系挖掘**：直接关系、多跳间接关系、同事与校友关系挖掘。
3. **合作价值评价**：基于 PageRank 与成果贡献度的双节点协同评价。
4. **全景态势展现**：科技企业统研、产业链宏观沙盘、大事件脉冲传播。
5. **产业-人才跨域大融合**：独立的异构建模展现，辅助硬科技背调与招商。
